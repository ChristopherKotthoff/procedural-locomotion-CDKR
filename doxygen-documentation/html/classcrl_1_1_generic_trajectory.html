Detailed Description

template<class T>
class crl::GenericTrajectory< T >

This class is used to represent generic trajectories. The class parameter T can be anything that provides basic operation such as addition and subtraction. We'll define a trajectory that can be parameterized by a 1d parameter (t). Based on a set of knots ( tuples <t, T>), we can evaluate the trajectory at any t through interpolation. Outside the range of the data, the closest stored value is returned.
