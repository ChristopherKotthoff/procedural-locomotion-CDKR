Detailed Description

This abstract class has a container for robots, rigid bodies and joints and can be considered as a "world" where every rbs lives. For physics simulation, derived class of RBEngine implements the physics world based on physics engine. For real robots, derived class of RBEngine corresponds to a network of robots.
