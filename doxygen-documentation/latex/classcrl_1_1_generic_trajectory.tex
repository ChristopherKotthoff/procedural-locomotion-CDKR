\doxysubsection{Detailed Description}
\subsubsection*{template$<$class \mbox{\hyperlink{im_gu_i_z_m_oquat_8cpp_ae53008cb938e8e14ae47291e41be8bec}{T}}$>$\newline
class crl\+::\+Generic\+Trajectory$<$ T $>$}
This class is used to represent generic trajectories. The class parameter T can be anything that provides basic operation such as addition and subtraction. We\textquotesingle{}ll define a trajectory that can be parameterized by a 1d parameter (t). Based on a set of knots ( tuples \texorpdfstring{$<$}{<}t, T\texorpdfstring{$>$}{>}), we can evaluate the trajectory at any t through interpolation. Outside the range of the data, the closest stored value is returned.
