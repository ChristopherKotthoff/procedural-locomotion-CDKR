\doxysubsection{Detailed Description}
This class is for loading Robot\+RB, \mbox{\hyperlink{classcrl_1_1loco_1_1_r_b_joint}{RBJoint}} and their properties from a various file formats. \mbox{\hyperlink{classcrl_1_1loco_1_1_r_b_loader}{RBLoader}} object is constructed when the new robot is created from file and destructed when the loading is finished.
