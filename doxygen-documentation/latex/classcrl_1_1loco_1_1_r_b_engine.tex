\doxysubsection{Detailed Description}
This abstract class has a container for robots, rigid bodies and joints and can be considered as a \char`\"{}world\char`\"{} where every rbs lives. For physics simulation, derived class of \mbox{\hyperlink{classcrl_1_1loco_1_1_r_b_engine}{RBEngine}} implements the physics world based on physics engine. For real robots, derived class of \mbox{\hyperlink{classcrl_1_1loco_1_1_r_b_engine}{RBEngine}} corresponds to a network of robots.
